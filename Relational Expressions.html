Relational Expressions

Example 1. Which of the following algebraic expresstions are polynomials?

i. x2 + 5x + 3     ii. x2 + 2√x

Solutions:

i.  x2 + 5x + 3 is a polynomial, since the power of x in each term is a
    non-negative integer.

ii.  x2 + 2√x is not a polynomial, since the power of x in second term
      2√x is 1/2 which is a fraction and not a non-negative integer.
