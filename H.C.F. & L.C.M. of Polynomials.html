H.C.F. & L.C.M. of Polynomials

Example 1.    Find the H.C.F. of the polynomials :

            p(x) = x2 - x - 6     and     q(x) = x3 - 27.

Solutions:

Factors of polynomials p(x) and q(x) are given below:

x2 - x - 6 = ( x - 3 ) ( x + 2 )

x3 - 27 = ( x - 3 ) ( x2 + 3x + 9 ) .

Since highest degree common divisor is ( x - 3 ) .

Hence H.C.F.= ( x - 3 )           Ans.
